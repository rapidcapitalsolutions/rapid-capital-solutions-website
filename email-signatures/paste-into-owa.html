Outlook web still shows “No signature yet”

Paste once into the roaming signature UI (this is what outlook.cloud.microsoft uses):
  1. Click Select all below, then Ctrl+C
  2. In Outlook: Settings → Account → Signatures → + Add signature
  3. Name it RCS, paste into the editor, Save
  4. Set both defaults to RCS
  5. New message — signature should appear

You were on submissions@ — do this while signed into that mailbox (and Vincent’s if you use that too).
